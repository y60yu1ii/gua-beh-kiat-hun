# Proposal: 通書來源新增暨計算精準化

## Status: In Progress

## Motivation

現有 gehuen 黃曆系統僅引用五部典籍，且計算邏輯有若干未嚴謹之處：
1. 僅《碎金賦》《象吉通書》《玉匣記》《協紀辨方書》《十傳通書》五部
2. 周堂、三刑等判斷邏輯未說明推算源頭
3. 建除十二值星黃黑道歸屬未標明
4. 伏吟、反吟等重要神煞未納入
5. 無典籍原文依據引用（僅口訣級別）

需新增通書來源並逐條核對計算精準度。

## Alternatives Considered

- **外部農曆庫依賴**：目前依賴 lunar-javascript CDN，該庫準確度已受廣泛驗證，保留
- **純手寫演算法**：工作量大且易錯，現有實作已驗證可行架構，不採用

## Decision

新增以下通書來源並逐項核對現有計算：
1. 《革除去津》— 嫁娶安床吉凶取用
2. 《選擇趨避通書》— 彭祖百忌、各日禁忌
3. 《永寧通書》— 伏吟反吟、紅砂大煞
4. 《協紀辨方書》（深化）— 十二值星黃黑道歸屬
5. 《天機蠡書》— 桃花、紅鸞等婚嫁吉神

逐項比對：典籍原文 → 口訣 → JS 實作 → 驗證是否符合。
